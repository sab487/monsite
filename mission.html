Épreuve E5 — Situations Professionnelles

Épreuve E5 — Support et mise à disposition de services informatiques : Cette épreuve évalue la capacité à prendre en charge des situations professionnelles réelles rencontrées en stage ou en entreprise. Chaque situation décrit un contexte, les tâches effectuées, les compétences mobilisées et les résultats obtenus.
MISSION E5 — N°1

Installation et configuration d'un serveur DHCP

Windows Server DHCP Réseau

Mise en place d'un serveur DHCP sous Windows Server pour l'attribution automatique des adresses IP sur le réseau de l'entreprise. Configuration des plages d'adresses, des exclusions et des réservations statiques pour les équipements critiques.

Consulter la fiche E5 Documentation
MISSION E5 — N°2

Sécurisation des accès par mise en place de la 2FA

Sécurité MFA Active Directory

Déploiement de l'authentification à double facteur sur les comptes utilisateurs du domaine Active Directory. Configuration de Microsoft Authenticator et rédaction d'une procédure utilisateur pour accompagner la migration.

Consulter la fiche E5 Documentation
MISSION E5 — N°3

Supervision du réseau avec Zabbix

Supervision Zabbix Linux

Installation et configuration d'un outil de supervision réseau Zabbix sur serveur Debian. Ajout des équipements à surveiller (switchs, serveurs, postes), création de tableaux de bord et paramétrage des alertes par email en cas d'incident.

Consulter la fiche E5 Documentation
MISSION E5 — N°4

Sauvegarde et restauration des données avec Veeam

Sauvegarde Veeam Virtualisation

Mise en place d'un plan de sauvegarde automatisé des machines virtuelles avec Veeam Backup. Configuration des planifications de sauvegarde, test de restauration et documentation du Plan de Reprise d'Activité (PRA) associé.

Consulter la fiche E5 Documentation
MISSION E5 — N°5

Déploiement d'un pare-feu pfSense

Firewall pfSense Sécurité réseau

Installation et configuration d'un pare-feu pfSense pour sécuriser le périmètre réseau de l'entreprise. Création de règles de filtrage, mise en place d'un VPN site-à-site et configuration de la journalisation des événements de sécurité.

Consulter la fiche E5 Documentation
MISSION E5 — N°6

Administration et durcissement d'un serveur Linux

Linux Debian SSH Durcissement

Sécurisation d'un serveur Debian en production : désactivation des services inutiles, configuration du pare-feu iptables, restriction des accès SSH par clé publique, mise en place de fail2ban et application des recommandations de durcissement de l'ANSSI.

Consulter la fiche E5 Documentation

Épreuve E6 — Projets SISR

Épreuve E6 — Parcours de professionnalisation : Cette épreuve repose sur deux projets techniques majeurs menés sur les deux années de BTS. Chaque projet fait l'objet d'un dossier complet décrivant le contexte, la démarche, les solutions mises en œuvre et les résultats obtenus.
PROJET E6 — MAJEUR

Projet E6 n°1 : Gestion de parc avec GLPI

GLPI FusionInventory Debian SQL

Mise en place d'une solution complète de gestion de parc informatique et de helpdesk. Installation du serveur GLPI sous environnement Debian, configuration de l'inventaire automatisé via FusionInventory et déploiement de la base de données SQL pour le suivi des actifs et la gestion des tickets d'assistance.

Consulter le dossier E6 Fiche situation
PROJET E6 — MAJEUR

Projet E6 n°2 : Infrastructure Active Directory & GPO

Windows Server Active Directory GPO Sécurité

Déploiement d'un contrôleur de domaine Windows Server pour la centralisation des ressources. Organisation de l'annuaire via des Unités Organisationnelles (UO), gestion des comptes utilisateurs et sécurisation du parc par des stratégies de groupe (GPO) dédiées au durcissement et à la restriction des accès.
